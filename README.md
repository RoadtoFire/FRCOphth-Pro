# FRCOphth Pro - Medical Quiz Platform

A comprehensive web application designed for medical professionals to prepare for FRCOphth examinations. This project was developed as a freelance solution, delivering a robust quiz engine with user performance tracking.

## 🔗 Live Demo
**Click here to view the live app:** [https://frcophth-pro-eu.onrender.com/](https://frcophth-pro-eu.onrender.com/)

## ⚡ Key Features
* **User Authentication:** Secure signup/login system.
* **Dynamic Quiz Engine:** Fetches questions from a PostgreSQL database with randomized ordering.
* **Progress Tracking:** Users can review past attempts and track scores over time.
* **Responsive Design:** Fully functional on desktop and mobile devices.

## 🛠 Tech Stack
* **Backend:** Django (Python)
* **Database:** PostgreSQL (Production), SQLite (Dev)
* **Frontend:** HTML5, CSS (Bootstrap), JavaScript
* **Deployment:** Render
